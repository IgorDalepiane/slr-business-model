# Context Repository

This folder contains all extracted context from the SLR repository, organized for AI-assisted article writing.

## Structure

- **`research_questions/`** - Research questions (RQ01-RQ10) definitions and mappings
- **`extraction_questions/`** - Extraction questions (EQ01-EQ33) details and organization
- **`closed_ended/`** - Closed-ended questions data, statistics, and distributions
- **`open_ended/`** - Open-ended questions data and extracted quotes
- **`coding_scheme/`** - Coding scheme (61 codes, 9 classes, 35 subclasses) and classifications
- **`prompts/`** - Prompts used during data extraction and analysis
- **`analysis/`** - Analysis reports, statistics, and key findings

## Source Files

All context points to specific source files in the repository:
- `by_closed_ended_question/` - Quantitative data source
- `by_open_ended_question/` - Qualitative data source
- `raw_data/` - Original paper data
- `prompts/` - Extraction prompts

## Usage

When writing article sections, reference these context files to:
1. Understand data structure and organization
2. Find specific statistics and distributions
3. Locate quotes and qualitative evidence
4. Map data to research questions
5. Support claims with source references
